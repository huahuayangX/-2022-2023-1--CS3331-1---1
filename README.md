### 作业要求：

在疫情期间，各个小区居民发挥互助精神，进行物品交换，互通有无。请你编写一个物品交换软件。
该程序允许添加物品的信息，删除物品的信息，显示物品列表，也允许查找物品的信息。

### 简介及使用

>* 本程序使用结构体数组存储物品数据。
>* 输入数字后回车，可选择对应功能。
>* 查找物品需要物品名称和主人姓名两个数据，输入后将得到物品名称、物品主人姓名、物品主人联系方式。查找到对应物品后可选择是否取出物品，若取出将删除数组中的物品数据。
